# Hannah Bennie Schools (HBS) WebApp

Official web platform for Hannah Bennie Schools, the No.1 NECTA-ranked Nursery and Primary school in Dar es Salaam Region.

## Features

- **Multi-language Support:** English and Kiswahili.
- **AI-Powered Recruitment:** Deepseek integration for job description generation and CV analysis.
- **Context-Aware Chat:** HBS Assistant for parent inquiries.
- **Admin Dashboard:** Management for Blogs, Events, Careers, and Applications.
- **Student Admissions:** Seamless inquiry and application flow.

## Tech Stack

- **Frontend:** Next.js (App Router), Tailwind CSS, Framer Motion.
- **Backend:** Supabase, Edge Functions.
- **AI:** Deepseek API.
- **Icons:** Lucide React.

## Getting Started

1. Install dependencies: `npm install`
2. Set environment variables in `.env.local` (see `.env.example`).
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
